Existem duas formas de obter um repositório git.

1. Na primeira forma, nos podemos pegar uma pasta no nosso computador e transformar ele em um repositório Git, ou
2. Clonar um repositório já existente

### Inicializar um repositório git em uma pasta existente

Se você tem um projeto existente na sua máquina e quer controlar a pasta com o Git primeiramente você precisa ir atá a pasta do diretório onde está o projeto.

Estando dentro da pasta do projeto, basta abrir um terminal e digitar o seguinte comando:

```bash
git init
```

Esse comando vai gerar dentro do diretório do projeto um subdiretório chamado `.git` , que contém todos os arquivos de repositório necessários.

### Clonando um repositório existente

Se você quiser baixar um projeto existente (clonar), o comando que vamos usar é o `git clone`. Esse comando não só baixa os arquivos do projeto mas também o histórico daquele projeto e automaticamente configura a conexão com o repositório remoto.

Para clonar um repositório, do github por exemplo, necessitamos do link daquele repositório e por fim rodar o comando:

```bash
git clone <url>
```
