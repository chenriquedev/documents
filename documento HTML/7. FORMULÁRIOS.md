Os formulários são usados para coletar dados inseridos pelo usuário.
Esses dados podem ser enviados a um servidor para processamento.

O elemento que vamos usar para criar um formulário é a tag `<form>`

**Exemplo:**

```html
<form>
	<!--Elementos de formulário aqui-->
</form>
```

O `<form>` é um container para diferentes tipos de elementos de input, como :
- Text fields
- Check boxes
- radio buttons
- submit buttons
- input element
- Outros elementos de entrada

### Campos de Texto
O `<input type="text">` define um campo de texto de uma única linha para entrada de dados.

**Exemplo:**

```html
<form>  
  <label for="fname">First name:</label><br>  
  <input type="text" id="fname" name="fname"><br>  
  <label for="lname">Last name:</label><br>  
  <input type="text" id="lname" name="lname">  
</form>
```

>[!TIP] **Dica:**  
O atributo ID e o elemento CLASS são bastante utilizados para CSS e manipulação do html com javascript.

> [!Warning] Atenção:
> O elemento `<label>` permite definir um rótulo para muitos elementos de formulário.
> Ele é útil para leitores de tela.
> O atributo **for** da tag `<label>` deve conter o mesmo valor do atributo **id** do elemento do input.


É assim que o código acima aparece no navegador.

<form>
  <label for="fname">First name:</label><br>
  <input type="text" id="fname" name="fname"><br>

  <label for="lname">Last name:</label><br>
  <input type="text" id="lname" name="lname">
</form>

### Checkboxes

O `<input type="checkbox">` define um checkbox.
Checkboxes permitem que o usuário escolha uma ou mais opções disponíveis.

**Exemplo**

```html

  <input type="checkbox" id="veiculo2" name="veiculo2" value="Carro">

  <input type="checkbox" id="veiculo3" name="veiculo3" value="Barco">
```

</form>

### O botão de submit
O `<input type="submit">` define um botão para submeter (enviar) os dados do formulário .

**Exemplo**

```html
<form >
    <label for="fname">Nome:</label><br>
    <input type="text" id="fname" name="fname" value="João"><br>

    <label for="lname">Sobrenome:</label><br>
    <input type="text" id="lname" name="lname" value="Silva"><br><br>

    <input type="submit" value="Enviar">
</form>
```

É assim que o código acima aparece no navegador.

<form >
    <label for="fname">Nome:</label><br>
    <input type="text" id="fname" name="fname" value="João"><br>

    <input type="text" id="lname" name="lname" value="Silva"><br><br>

    <input type="submit" value="Enviar">
</form>

### Atributo name

O atributo **name** serve para identificar o dado que foi digitado em um campo de formulário.
É o nome da informação que será enviada.

**Exemplo**

```html
<input type="text" name="usuario" value="Henrique">
<input type="text" name="idade" value="26">
```

Quando o formulário for enviado, o navegador manda os dados assim:

```bash
...?usuario=Henrique&idade=26
```

> [!Warning] Atenção:
> O name é o nome do campo que será enviado.
> Sem ele, o valor não é transmitido
