Tabelas no HTML permitem que você organize dados em linhas e colunas como se fosse uma planilha.

Para criar uma tabela, usamos uma combinação de tags que definem:
- A **tabela em si**: `<table>`
- As **linhas**: `<tr>` (table row)
- As **células de dados**: `<td>` (table data)
- As **células de cabeçalho**: `<th>` (table header)

**Exemplo:**

<table border="1">
	<tr>
		<th>Nome</th>
		<th>Telefone</th>
		<th>Endereço</th>
	</tr>
	<tr>
		<td>Maria</td>
		<td>8599999999</td>
		<td> Leste Oeste 1010</td>
	</tr>
	<tr>
		<td>José</td>
		<td>32140000</td>
		<td> General Sampaio 2020</td>
	</tr>
	<tr>
		<td>Antonio</td>
		<td>85988888888</td>
		<td> Praça do ferreira 2020</td>
	</tr>
</table>

```html
<table border="1">
	<tr> <!--linha-->
		<th>Nome</th> <!--cabeçalho-->
		<th>Telefone</th> <!--cabeçalho-->
		<th>Endereço</th> <!--cabeçalho-->
	</tr>
	<tr>
		<td>Maria</td> <!--Célula-->
		<td>8599999999</td> <!--Célula-->
		<td> Leste Oeste 1010</td> <!--Célula-->
	</tr>
	<tr>
		<td>José</td>
		<td>32140000</td>
		<td> General Sampaio 2020</td>
	</tr>
	<tr>
		<td>Antonio</td>
		<td>85988888888</td>
		<td> Praça do ferreira 2020</td>
	</tr>
</table>
```

> [!NOTE] Observação:
A tag `<table>` cria a tabela.
A tag `<tr>` cria uma linha (table row).  
 A tag `<th>` cria uma célula de **cabeçalho** (com texto em negrito e centralizado por padrão).
 A tag `<td>` cria uma célula de **dado comum**.

>[!TIP] **Dica:**  
Você pode ajustar a espessura da borda com o atributo `border`.  
Exemplo: `border="2"` cria bordas mais grossas.
