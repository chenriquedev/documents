No html possuímos diversos elementos. Como vimos antes os elementos html são definidos por meios das tags `<tagname> conteúdo </tagname>`.

### Tags de título:

As tags de título são definidas a partir da tag `<h1>` até a tag `<h6>`, sendo `<h1>` mais importante semanticamente do que as demais tags e a tag `<h6>` como menos importante.

##### Exemplos: 

<h1>Eu sou um título h1</h1>
<h2>Eu sou um título h2</h2>
<h3>Eu sou um título h3</h3>
<h4>Eu sou um título h4</h4>
<h5>Eu sou um título h5</h5>
<h6>Eu sou um título h6</h6>

```html
<h1>Eu sou um título h1</h1>
<h2>Eu sou um título h2</h2>
<h3>Eu sou um título h3</h3>
<h4>Eu sou um título h4</h4>
<h5>Eu sou um título h5</h5>
<h6>Eu sou um título h6</h6>

```

> [!NOTE]
>  Só devemos ter um único elemento h1 por página, pois ele representa o tema principal do conteúdo. Essa prática garante melhora a organização semantica, favorece o SEO (otimização para mecanismos de buscas, como o Google) e garante melhor acessibilidade.


### Tag de parágrafo

O elemento html  que define um parágrafo é a tag `<p>`. Um parágrafo sempre inicia em uma nova linha.

##### Exemplo

<p>Eu sou um parágrafo</p>
<p>Eu sou outro parágrafo</p>
```html
<p>Eu sou um parágrafo</p>
<p>Eu sou outro parágrafo</p>
```

### Tag de imagem

O elemento html que define uma imagem no nosso site é a tag `<img>`. Essa tag é uma das poucas tags que possui apenas a tag de abertura.

##### Exemplo:

```html
<img src="link da sua imagem" alt="descrição da imagem">
```

>[!WARNING] A tag `<img>` possui dois atributos **obrigatórios**:
>1. `src` (source): Indica o caminho (link ou arquivo) da imagem.
>2. `alt` (alternative text): É o **texto alternativo** que descreve a imagem.

**Por que o `alt` é tão importante?**

- **Acessibilidade:** Leitores de tela (usados por pessoas com deficiência visual) leem esse texto em voz alta.
    
- **Falhas:** Se a imagem falhar ao carregar (link quebrado, internet lenta), o texto do `alt` será exibido no lugar.
    
- **SEO:** O Google usa esse texto para entender o conteúdo da imagem.


### Tag de link

O elemento html que define um link no nosso site é a tag `<a>`. Um elemento de link, ou hyperlink, não necessariamente pode ser um texto, ela pode ser uma imagem, por exemplo.

##### Exemplo: 

```html
<a href="https://www.google.com">Visite o google</a>
<!--Podemos envolver um elemento html com a tag a-->
<!--Nesse caso eu estou colocando uma imagem dentro da tag a-->
<a href="https://www.google.com"><img src="link da sua imagem" alt=""></a>
```
