
Um elemento HTML pode ter atributos. Esses atributos fornecem informação adicional sobre os elementos. Atributos do elemento sempre serão inseridos na tag de abertura. Os atributos geralmente são formados por pares de chave e valor, exemplo: chave="valor"

**Exemplo:** 

```html
<img src="" alt="">
```

Neste exemplo temos a tag img. A tag img possui 2 atributos essenciais, o src e o alt.

- O atributo **src** define o caminho da imagem
	- **Exemplo:** src="./minha_imagem.png"
	
-  O atributo **alt** define um texto alternativo para aquela imagem, que vai aparecer caso a imagem não carregue ou o usuário utilize leitores de tela
	- **Exemplo:** alt="Gato laranja dormindo em cima de uma mesa"


**Boa parte dos elementos html possuem atributos**

**Exemplo:**

```html
<html lang="pt-br"> <!--Lang define o idioma da página-->
<a href=""></a> <!--href define a URL da página para onde o link redireciona-->
<p title="isso é um parágrafo"></p> <!--O atributo title define uma informação extra sobre o elemento html. O valor aparece quando passamos o mouse por cima-->
```

**Exemplo do uso do atributo title:**
<p title="Isso é uma dica que aparece no hover!">Passe o mouse por cima de mim para ver a mágica!</p>
