
### Elementos de bloco

Elementos de bloco (block-level) sempre vão começar em uma nova linha.
Os navegadores adicionam automaticamente um espaço antes e depois desses elementos.

Esses elementos ocupam toda a largura  disponível da tela (ou do container).

**Exemplo:**

<h1 style="border: 1px solid white; padding-top: 2px; padding-bottom: 2px">Eu sou um título H1 </h1>
<p style="border: 1px solid white; padding-top: 2px; padding-bottom: 2px">Eu sou um parágrafo </p>

> [!NOTE] Observação:
> Note que, independente do tamanho do texto, a "caixa" do elemento ocupa toda a largura da página

### Elementos de linha (inline)

Elementos de linha (inline) não iniciam em uma nova linha.
Ao contrário de elementos de bloco, eles ocupam apenas o espaço necessário para o seu conteúdo, permitindo que outros elementos fiquem ao lado.

Eles são usados geralmente para destacar partes pequenas do texto, como links, palavras em negrito ou trechos coloridos, sem quebrar a linha.

**Exemplo:**

<a style="border: 1px solid white; padding: 3px" href="https://www.google.com">Visite o google</a> <a  style="border: 1px solid white; padding: 3px" href="https://www.youtube.com">Visite o youtube</a>

```html
<!--Exemplo acima-->
<a href="https://www.google.com">Visite o google</a> 
<a href="https://www.youtube.com">Visite o youtube</a>
```

Outro exemplo é a utilização dessas tags dentro de elementos de bloco, exemplo

<p style="border: 1px solid white; padding-top: 2px; padding-bottom: 2px">Eu sou um parágrafo com <span style="font-weight: bolder">um span</span> </p>

```html
<!--Exemplo acima-->
<p style="border: 1px solid white; padding-top: 2px; padding-bottom: 2px">Eu sou um parágrafo com <span style="font-weight: bolder">um span</span> </p>
```

> [!Warning] Atenção:
> O atributo STYLE é utilizado para estilizar nossos elementos HTML. Falaremos mais sobre ele no módulo de CSS

### Containers

Um "container" em HTML é um elemento que agrupa outros elementos, como texto, imagens e vídeos, para dar estrutura e organização. 
Eles servem como "caixas organizadoras" do conteúdo.

Usamos containers para:
- Organizar o conteúdo em seções. (ex: cabeçalho, rodapé)
- Aplicar estilos 
- Controlar o posicionamento e layout dos elementos internos
- Para facilitar a responsividade

**Exemplo:**

<div style="border: 2px solid white; padding: 10px;">
  <h1>Meu Site</h1>
  <p>Bem-vindo ao meu site!</p>
</div>


```html
<div style="border: 2px solid white; padding: 10px;">
  <h1>Meu Site</h1>
  <p>Bem-vindo ao meu site!</p>
</div>
```

Nesse exemplo a tag `<div>` funciona como um container que agrupa o título e o parágrafo.

**Exemplo com múltiplos containers:**

```html
<!--Nesse exemplo separamos o conteúdo em 3 partes lógicas-->
<html lang="pt-br">
	<head>
	</head>
	<body>
	<!--Aqui temos a parte superior do site, geralmente o menu de navegação-->
	  <div>
	    <h1>Topo do site</h1>
	    <p>Aqui vai o nome ou logo do site.</p>
	  </div>
	<!--Aqui temos a parte que vai a parte do conteúdo do site-->
	  <div>
	    <h2>Conteúdo principal</h2>
	    <p>Esse é o texto central da página.</p>
	  </div>
	  <!--E aqui temos o rodapé do site -->	
	  <div>
	    <h3>Rodapé</h3>
	    <p>© 2025 - Todos os direitos reservados</p>
	  </div>
	
	</body>
</html>
```

>[!TIP]  Dica
Mesmo sem CSS, o uso de containers já ajuda a **organizar visualmente e logicamente o conteúdo** da página.  
Em aulas futuras, você pode **adicionar estilo** a essas áreas com CSS e ver como cada container pode ter sua própria aparência.
