
O **HTML**, ou Hypertext Markup Language, é uma linguagem de marcação utilizada para
criar e estruturar nossos sites.

O html consiste de uma série de elementos e esses elementos dizem ao browser como mostrar o conteúdo.
#### Exemplo básico de um documento HTML

```html
<!DOCTYPE html> <!--Essa declaração define que isto é um documento HTML5-->
<html> <!--Elemento "pai" de uma página HTML-->
    <head> <!--Elemento que contém meta informações sobre a página HTML-->
        <title>Page Title</title> <!--Elemento responsável por especificar o título da página html-->
    </head>
    <body> <!--Elemento que define o corpo do documento. Aqui vai conter todo o conteúdo visível ao usuário-->
        <h1>My First Heading</h1> <!--Elemento que define um título-->
        <p>My first paragraph.</p> <!--Elemento que define um parágrafo-->
    </body>
</html>
```

> [!NOTE] Dica
> Para gerar o "esqueleto" do nosso arquivo html basta digitar " ! " no vscode e apertar enter

> [!NOTE] Informação
> Toda página HTML deve conter as tags `<html>, <head> e <body>`


## O que é um elemento html ?

Um elemento HTML é um componente fundamental da estrutura de uma página web. Ele é geralmente definido por 3 partes

1. **Tag de abertura:** (ex: `<p>`)
2. O **conteúdo:** (ex: `Olá mundo!!!`)
3. **Tag de fechamento:** (ex: `</p>`)

> Nem todas tags do HTML tem a **tag de fechamento**, por exemplo a tag `<img>`

#### Exemplo correto

```html
<h1>Aqui temos um título</h1>
<p>Aqui temos um parágrafo</p>
<!--Aqui temos a declaração de uma tag de imagem-->
<img src="" alt="">
```

> Sempre verificar se a tag está corretamente fechada.



> [!WARNING] Erros comuns
> 
> É muito comum esquecer de fechar tags ou fechar a tag de abertura incorretamente.



```html
<h1 Aqui está faltando fechar a tag de abertura</h1>
<p>Aqui esta faltando a tag de fechamento
```

>[!TIP] Correção
Preste sempre atenção se a tag de abertura foi fechada com `>` e se a tag de fechamento `</tagname>` existe no final.

```html
<h1> Aqui está o fechamento correto da tag de abertura</h1>
<p>Aqui a tag de fechamento foi incluída.</p>
```
