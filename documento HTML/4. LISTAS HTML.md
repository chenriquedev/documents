Listas html permitem que você agrupe um conjunto de itens relacionados em uma lista

### Lista não ordenada

A lista não ordenada se inicia com a tag `<ul>` e cada item da lista é uma tag `<li>`.

<ul>
	<li>item</li>
	<li>item</li>
	<li>item</li>
</ul>

```html
<ul>
	<li>item</li>
	<li>item</li>
	<li>item</li>
</ul>
```

### Lista ordenada

A lista  ordenada se inicia com a tag `<ol>` e cada item da lista é uma tag `<li>`.

<ol>
	<li>Primeiro item</li>
	<li>Segundo item</li>
	<li>Terceiro item</li>
</ol>

```html
<ol>
	<li>item</li>
	<li>item</li>
	<li>item</li>
</ol>
```


### Lista descritiva

No html também possui um formato de lista que possui descrições entre os itens. Esse elemento é declarado com a tag `<dl>`, cada item dessa lista é declarado com a tag `<dt>` e cada descrição é declarado com a tag `<dd>`

<dl>
	<dt>Café</dt>
	<dd>- Bebida quente</dd>
	<dt>Leite</dt>
	<dd>- Bebida gelada</dd>
</dl>

```html
<dl>
	<dt>Café</dt>
	<dd>- Bebida quente</dd>
	<dt>Leite</dt>
	<dd>- Bebida gelada</dd>
</dl>
```
