#### Desafio: Página "Sobre Mim"

Objetivo: Combinar títulos, parágrafos, listas e links.

Crie uma nova página HTML chamada `sobre.html`. Ela deve ter:

1. A estrutura básica (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
    
2. Um título na aba do navegador (`<title>`).
    
3. O título principal da página, que deve ser **apenas um** `<h1>`.
    
4. Um parágrafo (`<p>`) se apresentando.
    
5. Uma **lista não ordenada** (`<ul>`) com seus 3 hobbies favoritos (cada um em um `<li>` ).
    
6. Um **link** (`<a>`) que leve para o Google, com o atributo `href` preenchido

#### Desafio: Tabela de Horários

Objetivo: Estruturar dados usando tabelas.

Crie uma página chamada `horarios.html` que mostre uma tabela de horários de estudo.

1. Crie a tabela com a tag `<table>`.
    
2. Use o atributo `border="1"` para que ela tenha bordas..
    
3. A primeira linha (`<tr>`) deve ser o **cabeçalho** (`<th>`), com as colunas: "Hora", "Segunda-feira" e "Terça-feira".
    
4. Adicione pelo menos duas linhas de dados (`<tr>`) usando células de dados (`<td>`) para preencher os horários.

#### Desafio: Formulário de Contato Completo

Objetivo: Usar formulários, labels e inputs diferentes (foco em acessibilidade).

Crie uma página `contato.html` com um formulário de contato.

1. Inicie o formulário com a tag `<form>`.
    
2. Crie um campo para "Nome" e um para "Email", ambos `input type="text"`.
    
3.  Para cada campo, adicione uma `<label>`. O atributo `for` da label deve ser **igual** ao `id` do input.
    
4. Adicione a pergunta "Qual o motivo do contato?" usando **Radio Buttons** (`input type="radio"`). Dê duas opções (ex: "Dúvida", "Sugestão").
    
    - _Lembrete:_ Para que o usuário só possa marcar uma, ambos os inputs de rádio devem ter o **mesmo** atributo `name` (ex: `name="motivo"`).
        
5. Adicione um **Checkbox** (`input type="checkbox"`) com a `<label>` "Aceito receber novidades por e-mail".
    
6. Finalize com um botão de envio (`<input type="submit" value="Enviar">`)
